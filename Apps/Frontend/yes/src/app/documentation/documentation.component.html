Introduction

Welcome to the documentation section of Mania Credit Checker! Mania Credit Checker is a powerful tool designed to assist you in analyzing credit-related datasets efficiently.

Getting Started

To begin using Mania Credit Checker, follow these steps:

  1. Upload your dataset using the provided input field. Mania Credit Checker supports datasets in CSV format.
  2. View existing datasets or upload a new one.
  3. Perform various actions such as deleting, showing, treating, and analyzing datasets.

Features

Mania Credit Checker offers the following features:

  • Dataset Management: Upload, manage, and analyze credit-related datasets effortlessly.
  • Analytics: Obtain valuable insights from your datasets, including missing values count and numeric statistics.
  • Data Treatment: Treat your datasets with various operations to prepare them for analysis.
  • User-friendly Interface: Intuitive design and user-friendly interface make it easy to navigate and interact with the application.

Usage

To use Mania Credit Checker, follow these steps:

  1. Upload Dataset: Select a CSV file containing your credit-related data and click on "Upload Dataset" to upload it to the application.
  2. View Datasets: Explore existing datasets or upload a new one using the provided buttons.
  3. Analyze Dataset: Click on "Analytics" to perform analysis on the uploaded dataset and view insights such as missing values count and numeric statistics.
  4. Perform Actions: Utilize the action buttons to perform various operations on datasets, including deletion, showing, treating, and analyzing.
